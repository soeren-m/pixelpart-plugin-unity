# Getting Started {#mainpage}

This is the documentation of the *Pixelpart* plugin for the Unity game engine. Start by [installing the plugin](./installation.md) and [use Pixelpart effects](./usage.md) in your game. The plugin also offers an extensive [scripting API](./scripting.md) to dynamically adjust effects from C# scripts.

## Requirements

Unity 2021.3 or higher is required to use the Pixelpart plugin.

## Platform Support

The plugin supports the following platforms:

Platform | Architectures | Supported
-------- | ------------- | ---------
Windows | x64, x86_32 | Yes
Linux | x64, x86_32 | Yes
macOS | Universal | Yes
iOS | arm64 | Yes
Android | arm64, arm32, x64, x86_32 | Yes
Web | wasm32 | Yes
Console | | No

## Feature Support

The plugin supports the following features:

Feature | Supported
------- | ---------
2D effects | Yes
3D effects | Yes
Inputs | Yes
Triggers | Yes
Events | Yes
Custom materials | Unlit materials only
